⚙️ Implementation Logic:
Backend:
Track active connections using:
WebSockets (preferred) OR
Server-Sent Events (SSE)
Increment viewer count when:
User opens stream
Decrement when:
User disconnects or leaves
// Example logic
let viewers = 0;

io.on("connection", (socket) => {
  viewers++;
  io.emit("viewerCount", viewers);

  socket.on("disconnect", () => {
    viewers--;
    io.emit("viewerCount", viewers);
  });
});
⚡ Frontend:
Subscribe to real-time updates
Render dynamically without reload

  setViewerCount(count);
